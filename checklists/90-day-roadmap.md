# 90-Day Roadmap — Track B (AI Transformation)

## Month 1 — Foundation & Audit (Days 1-30)

### Week 1-2: Layer 1 — Convert
- [ ] Complete Week 1 Sprint (`checklists/week1-sprint.md`)
- [ ] Deploy AI Audit / Assessment Generator on your website
- [ ] Set up nurture sequence in Brevo/MailerLite (4 emails from `templates/nurture/`)
- [ ] Deploy on-site chatbot (Tidio, manychat, or custom — free tier)
- [ ] Build basic landing page for the assessment tool

### Week 2-3: Technical SEO — Layer 3
- [ ] Run `tools/setup-technical-seo -o output/technical-seo`
- [ ] Deploy `robots.txt` and `llms.txt` at root of your domain
- [ ] Ensure your site uses Server-Side Rendering (SSR) — test with curl
- [ ] Install JSON-LD schemas (`templates/schema/`) on relevant pages
- [ ] Run baseline SoM audit using `tools/track-som`
- [ ] Submit sitemap to Google Search Console + Bing Webmaster Tools

### Week 2-4: LinkedIn Launch — Layer 2
- [ ] Set up LinkedIn Company Page (complete all fields)
- [ ] Begin Mon/Wed/Fri posting cadence using `templates/linkedin/`
- [ ] Generate first month of posts using `tools/generate-content`
- [ ] Start daily engagement (15 min/day commenting on target audience posts)
- [ ] Identify 30-50 target prospects on LinkedIn

## Month 2 — Acceleration (Days 31-60)

### Layer 2 — Visibility
- [ ] Launch cold email outreach wave (`tools/generate-outreach`)
- [ ] Register on global directories (`templates/directory-listings/global-listings.md`)
- [ ] Register on Singapore directories (`templates/directory-listings/sg-listings.md`)
- [ ] Publish first 2 SEO blog posts using `templates/seo/long-tail-post.md`
- [ ] Cross-post to Medium, Hashnode, Substack for backlinks
- [ ] Answer 3-5 questions on Reddit (r/singapore, r/askSingapore) and Quora
- [ ] Join relevant LinkedIn Groups and participate genuinely

### Layer 3 — AI Discoverability
- [ ] Rewrite top 3 service pages with answer-first blocks (`templates/seo/answer-block.md`)
- [ ] Add data tables and statistics to all pages (+37-40% citation lift)
- [ ] Earn first piece of earned media (HARO / Qwoted journalist query)
- [ ] Begin PR distribution (1 press release via PRLog or OpenPR)
- [ ] Monitor SoM weekly and record in tracker

## Month 3 — Maturation (Days 61-90)

### Layer 2 — Partnerships & PR
- [ ] Identify 5-10 non-competing Singapore firms for co-webinars
- [ ] Research partnership outreach targets
- [ ] Send partnership proposals (co-content, referral deals)
- [ ] Apply to speak at SGTech / AI Singapore events
- [ ] Publish 4-6 SEO blog posts total
- [ ] Keep directory profiles up to date and add more

### Layer 3 — Measurement & Refinement
- [ ] Monthly SoM report — compare week-over-week trends
- [ ] Analyse which queries/engines cite you most
- [ ] Identify gaps: which keywords get 0 citations?
- [ ] Double down on what works: create more content on cited topics
- [ ] Check competitor SoM — where are they strong that you're weak?

### Layer 1 — Optimise
- [ ] Review lead quality from assessment tool
- [ ] A/B test assessment form length and question order
- [ ] Review nurture sequence conversion rate (email → booked call)
- [ ] Refine chatbot conversation flow based on real interactions
- [ ] Set up lead scoring in Sheets

## Ongoing (Every week)

- [ ] 3 LinkedIn posts (Mon insight, Wed tip, Fri case study)
- [ ] 15 min/day LinkedIn engagement
- [ ] 3-5 cold emails / LinkedIn connection requests
- [ ] Monitor HARO/Qwoted for journalist queries
- [ ] Track SoM (weekly, 15 min)
- [ ] Log all activity and results
